🧭 각 섹션의 역할(한 줄 요약)
00100_SettingsLegend.vue — 페이지 상단에 “실제동작 / 로컬저장 / 준비중” 범례를 표시

00200_AccountSection.vue — 비밀번호 변경(모달 트리거), 친구신청 허용 토글, 회원 탈퇴(준비중)

00300_NotificationSection.vue — 푸시/메시지/친구신청 알림 토글 3종

00400_DisplaySection.vue — 글자 크기(ion-range), 테마(ion-select) 즉시 반영

00500_BillingSection.vue — 구매하기 버튼(향후 결제/구독 플로우 연결)

00600_SupportLegalSection.vue — 문의/공지/법적고지 버튼(향후 라우팅 연결)

00700_AppInfoSection.vue — 앱 버전/빌드 정보, (선택) 오픈소스 라이선스 링크

00800_PrivacySection.vue — (필요 시) 프로필 공개 범위/차단 옵션 등 개인정보 관련

00900_DataManagementSection.vue — 캐시 초기화/임시파일 정리 등

01000_BackupRestoreSection.vue — 설정 백업·복원

01100_AdvancedSection.vue — 실험실/개발자 옵션(토글 모음)

01200_DebugSection.vue — 로그 수집/내보내기/진단(내부용)
