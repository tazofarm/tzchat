
1. 로컬 테스트

front : 개발 pc local
back : 개발 pc local

명령어 : npm run dev

환경 값 : .env.development


2. 웹 배포

front : 서버 pc 
back : 서버 pc

명령어 : npm run build:web

환경 값 : .env.web


3. 사전 앱테스트 (로컬-서버)

front : 개발 pc local
back : 서버 pc

명령어 : npm run dev:remote

환경 값 : .env.dev-remote

4. 앱 배포

front : 휴대폰 앱
back : 서버 pc

명령어 : npm run build:app

환경 값 : .env.production
